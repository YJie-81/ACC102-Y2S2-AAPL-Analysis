# Apple Financial Ratio Analysis Using WRDS
## ACC102 Python Data Product - Track 2

### Project Purpose
This project uses Python and WRDS Compustat data to analyse Apple’s profitability, liquidity, and solvency ratios for accounting and financial analysis.

### Dataset
- Source: WRDS Compustat
- Company: Apple Inc. (AAPL)
- Access Date: 16 April 2026
- Variables: Revenue, Net Income, Assets, Liabilities

### Tools
- wrds, pandas, matplotlib

### How to Run
1. Install packages: pip install wrds pandas matplotlib
2. Enter your WRDS username
3. Run the notebook
